# Установка и первый запуск
```bash
# Установить
$ make docker-setup

# Запустить контейнеры
$ make compose-up

# Дождаться полного запуска и выполнить
$ make compose-migrate
$ make compose-link
```

# Запуск
```bash
$ make compose-up
```

Сервер будет доступен http://localhost:8000/
